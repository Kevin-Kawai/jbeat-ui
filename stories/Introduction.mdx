# jbeat-ui

A minimal CSS-only UI system for blogs.

## Overview

jbeat-ui is a lightweight, easy-to-use CSS framework that provides beautiful default styles for HTML elements. No JavaScript, no build step required for usage - just import the CSS and start writing semantic HTML.

## Features

- **CSS-only**: No JavaScript dependencies
- **Minimal**: Only ~1.6KB minified
- **Semantic**: Works directly with HTML elements
- **Customizable**: Built with CSS custom properties
- **Modern**: Uses modern CSS features

## Installation

```bash
npm install jbeat-ui
```

## Usage

### Via NPM/Bundler

```javascript
import 'jbeat-ui/dist/jbeat-ui.css';
```

### Via HTML

```html
<link rel="stylesheet" href="path/to/jbeat-ui.css">
```

## What's Included

- **Reset**: Modern CSS reset for consistent cross-browser rendering
- **Typography**: Beautiful heading and paragraph styles
- **Lists**: Styled ordered and unordered lists
- **Structure**: Layout components like articles and headers
- **Variables**: Customizable CSS custom properties
